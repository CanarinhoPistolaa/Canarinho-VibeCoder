# Contribuindo

## Setup local

```bash
git clone https://github.com/CanarinhoPistolaa/Canarinho-VibeCoder.git
cd Canarinho-VibeCoder
./install
```

Requer Node.js >= 22.

## Desenvolvimento

```bash
npm run build    # compila TypeScript
npm test         # roda testes
```

## Branches

- `feat/*` — novas features
- `fix/*` — bug fixes
- `docs/*` — documentação

## PRs

- Abra sempre contra `main`
- Preencha o template de PR
- CI precisa passar (tests + build)
